# pozo

Segundo juego mínimo (gate de abstracción D-8). Solo importa engine + mesh
vía presets/rooms.

```bash
npm run demo:pozo
```

Incluye dominio puro, autoridad, MCP jugador, vista (view-kit) y
`spec/CASOS.md`. README: `packages/games/pozo/README.md`.

```bash
npm test -w @zeus/pozo
npm run e2e:pozo-mcp
```
