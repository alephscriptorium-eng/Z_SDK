# View kit — `@zeus/view-kit`

Kit de vistas **3D + HTML** browser-safe. Escena, HUD, paneles, stick puppets
y wiring de room. Lo específico del juego (escenario, intents de dominio)
vive junto a las vistas del juego.

Consumidores actuales: `arg-console`, `3d-monitor`, `player-3d-ui`, vistas de
pozo. README: `packages/engine/view-kit/README.md`.
