# Playbook kit — `@zeus/playbook-kit`

Método **CASOS** como producto del SDK (no un extra del primer juego):

1. Formato de caso (precondición / pasos MCP / observación humana / criterio / errores)
2. Test de coherencia del playbook markdown
3. Plantilla de acta (`VALIDACION.md`)
4. Runner de la mitad MCP-verificable (pre-rellena el acta)

La mitad visual sigue siendo humana. Sin nombres de juego en el kit: el juego
aporta la ruta del playbook, el patrón de tools y el MCP HTTP.

Ver también la guía del [método playbook](/playbook/).
README: `packages/engine/playbook-kit/README.md`.
