# `@zeus/room-client-browser`

Cliente de room para navegador (scriptorium `/runtime`: `SET_STATE` +
`ROOM_MESSAGE`). Pareja browser de `@zeus/rooms` (Node).

```js
import { /* room client */ } from '@zeus/room-client-browser';
```

Subpaths: `./browser`, `./dev-config`.

## Tests

`npm test -w @zeus/room-client-browser`
